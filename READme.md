## Oversikt

Dette er et **Netflix-inspirert skjema** jeg lagde som skoleoppgave.  
Alt er bygget med HTML og CSS, og målet var å lære mer om skjemaer og design.

## Oppgavebeskrivelse

Vi skulle lage en side som lignet på Netflix, med et skjema der man fyller inn info.  
Kravene var blant annet:

- Bruke et bakgrunnsbilde over hele siden
- Ha en mørk boks oppå bildet
- Lage et skjema med input, select, radio, checkbox osv.
- Style skjemaet i Netflix-stil (rød knapp, hvit tekst, mørk boks)

## Bygget med

- HTML
- CSS

## Funksjoner

- Helsides bakgrunnsbilde
- Skjema med forskjellige input-felter
- Responsivt design med media queries
